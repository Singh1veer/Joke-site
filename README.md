# Joke of the Day 🎤

A server-side rendered web app that fetches categorized jokes from the JokeAPI v2 REST API.

## Tech Stack
Node.js · Express.js · EJS · Axios · Bootstrap 5

## Features
- Filter by category (Programming, Dark, Pun, Misc, Spooky, Christmas)
- Filter by joke type (Single / Two-Part)
- Server-side content filtering — blocks nsfw, racist, sexist, explicit jokes
- Graceful error handling on API failure

## Run Locally
Bash:
git clone https://github.com/Singh1veer/Joke-site.git
cd Joke-site
npm install
node index.js
# Open http://localhost:3000
